# Faction Folklore

Faction Folklore is a collection of five self-contained games set in the
MK-Dirge universe. Each game is designed to fit entirely within the capacity
of a standard 1.44 MB floppy disk while standing on its own as a complete
experience.

## Games

- **Wrath Blast** — Dungeon Crawler
- **Compulsive Cannon** — Puzzle Platformer
- **Seek & Destroy** — Survival Platformer
- **Steam Radiation** — Resource Horde Survival
- **Arena King** — Simplified MK-Dirge Battle Card Game

Each of the first four games awards an unlock code upon completion. Those
codes can be entered into **Arena King** to unlock powerful faction-themed
decks inspired by the challenges completed in the other games.

## Goals

- Native executables
- Cross-platform (Windows, macOS, Linux)
- No external dependencies
- Designed to fit on a 1.44 MB floppy disk
- Open source under the MIT License
